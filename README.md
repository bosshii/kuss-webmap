# kuss-webmap

## Project setup

```
# yarn
yarn

# npm
npm install
```

## Compiles and hot-reloads for development

```
# yarn
yarn dev

# npm
npm run dev
```

## Compiles and minifies for production

### Numbering versions automatically
Bash required
```
# yarn
yarn build

# npm
npm run build
```
### Number manually
Need to update .env before type
```
# yarn
yarn manual-build

# npm
npm run manual-build
```

## Check before publish

```
# yarn
yarn preview

# npm
npm run preview
```
